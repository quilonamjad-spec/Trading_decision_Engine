# 📈 Trade Decision Engine (TDE)

An intelligent stock decision support dashboard built with Python and Streamlit.

---

## Vision

Trade Decision Engine is designed to help traders identify the best opportunities from a watchlist by combining multiple technical indicators into a single explainable score.

Instead of manually checking dozens of charts, TDE answers one simple question:

> **"Which stock deserves my attention right now?"**
